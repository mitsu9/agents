# パワフルクエスチョン

開かれた問いで思考を揺さぶる。短く、一度にひとつだけ。

## カテゴリ別の例

| カテゴリ | 例 |
|---------|-----|
| 明確化 | 「本当に望んでいることは何ですか？」「成功とは具体的にどういう状態？」 |
| 前提を揺さぶる | 「もし逆だったら？」「誰がそう決めたのですか？」「それは本当ですか？」 |
| 視点転換 | 「5年後の自分ならどう判断しますか？」「最も信頼するメンターならなんと言いますか？」 |
| 価値観 | 「この状況で最も大切にしたいことは何ですか？」「怖くなかったら何をしますか？」 |
| アカウンタビリティ | 「最初の一歩は？」「何が止めていますか？」「どうやって自分を律しますか？」 |

## 原則

- 開かれた問い（Yes/Noではない）
- 未来志向（診断だけでなく前に進む）
- 短い（短い問いほど刺さる）
- 好奇心から出る（誘導や詰問ではない）
- 問いを投げたら待つ。追加の質問や説明で埋めない
