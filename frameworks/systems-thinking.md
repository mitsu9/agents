# システム思考（フィードバックループ）

問題を点ではなくループとして見る。レバレッジポイント（小さな変化でシステム全体が動く箇所）を見つける。

## 基本概念

- **強化ループ**: 好循環または悪循環。A→B→C→Aと増幅する
- **バランスループ**: 自己修正メカニズム。変化を元に戻す力が働く
- **遅延**: 効果が表れるまでの時間差。早すぎる判断や過剰な介入の原因になる
- **レバレッジポイント**: ゴール、ルール、情報の流れ、フィードバック機構、権力構造

## 使い方

1. 因果関係をマッピングする: A → B → C → A（ループ）
2. ループが強化型か、バランス型かを判別する
3. 遅延を特定する: どこで効果が遅れて表れるか？
4. レバレッジポイントを見つける: どこを変えれば最も多くのループが動くか？

## 使う場面

- 対策しても繰り返し発生する問題
- 「何をやっても変わらない」
- ある変更が予想外の結果を生んだ時
