# 人事評価の検討スキル

公正で多角的な人事評価を行うための思考セッションを開始します。

## 実行手順

### 1. 基盤の読み込み
- `context/shared.md`
- `personas/` (3ファイル)
- `frameworks/hr-evaluation.md`
- `templates/hr-evaluation.md`

### 2. 評価セッションの実施
1. **評価の目的と基準の確認**: 何を評価するのか、基準は何か
2. **事実の収集**: 成果・実績、具体的なエピソード（STAR法）、360度フィードバック
3. **多面的な評価**: 成果、行動、能力、態度の4つの観点から評価
4. **強みと改善点の特定**: 優れている点、改善が必要な点
5. **総合評価と判断**: 各評価基準のバランスを取る
6. **育成計画の策定**: 成長目標とアクションプラン

### 3. バイアスへの注意
ハロー効果、新近効果などの認知バイアスに注意し、公平性を確保

### 4. ペルソナからのフィードバック
各視点から評価を多角的に検討

### 5. セッションの保存
`{peoples}/[対象者名]/YYYY-MM-DD_人事評価_[対象者名].md` に保存
