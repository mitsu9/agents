# 半期目標作成スキル

半期目標を設定する。目標カスケードの最上位にあたる。

## 実行手順

### 1. 前半期の振り返り

前半期の目標ファイル（`Plan/YYYY-H1.md` または `Plan/YYYY-H2.md`）があれば読み、振り返りセクションを確認する。

### 2. 現在の文脈の把握

以下を必要に応じて読み込み、目標設定の材料とする:

- `{self-info}`: 役割・ポジション
- `{challenges}`: 現在の課題認識
- `{thinking}/Documents/general/思考キュー.md`: 考えたいことのストック

### 3. 半期目標ファイルの作成

`templates/half-year-goal.md` をベースに `Plan/YYYY-H1.md`（上半期）または `Plan/YYYY-H2.md`（下半期）を作成する。

### 4. 目標設定

`/think-goal` の活用を提案し、ユーザーと対話しながら目標を設定する:

- 2〜3個の目標に絞る
- 各目標に具体的な達成基準を設定する
- 半期を通じた優先順位を明確にする

### 5. 月へのブレークダウンの提案

半期目標が設定できたら、当月の月目標作成（`/plan-month`）を提案する。
