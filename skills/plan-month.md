# 月目標作成スキル

月目標を半期目標からブレークダウンして設定する。

## 実行手順

### 1. 半期目標の確認

当半期の目標（`Plan/YYYY-H1.md` または `Plan/YYYY-H2.md`）を読む。存在しない場合は、`/plan-half-year` での作成を提案する。

### 2. 前月の振り返り

前月の月目標（`{goals-month}`）があれば読み、振り返り・実績セクションを確認する。前月の振り返りが未記入の場合は、先に振り返りを行うか確認する。

### 3. 月目標ファイルの作成

`templates/monthly-goal.md` をベースに `{goals-month}` を作成する。

### 4. 目標設定

ユーザーと対話しながら目標を設定する:

- 半期目標のうち、当月に注力すべきものを特定する
- 2〜3個の目標に絞る
- 各目標に具体的な達成基準を設定する
- 目標が深い検討を要する場合は `/think-goal` の活用を提案する

### 5. 振り返り・実績セクションの説明

月末に振り返り・実績を記録するためのセクションがあることをユーザーに伝える。このセクションは評価時の実績記録として機能する。
